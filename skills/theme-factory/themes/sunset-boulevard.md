# Sunset Boulevard

温暖、鲜艳的日落色，带来戏剧性与活力。

## Color Palette

- **Sunset Orange**：`#ff7043` - 主色
- **Neon Pink**：`#ff4f81` - 高亮
- **Golden Glow**：`#ffcc66` - 次级强调
- **Charcoal**：`#2f2f2f` - 文本/对比底

## Typography

- **Headers**：Montserrat SemiBold
- **Body Text**：Source Sans Pro

## Best Used For

活动宣传、创意提案、娱乐/媒体 deck、需要高能量的展示。
