# Midnight Galaxy

戏剧化的深色主题，带有宇宙与星际氛围。

## Color Palette

- **Cosmic Black**：`#0b0d21` - 背景
- **Nebula Purple**：`#673ab7` - 重点
- **Starlight Blue**：`#3f51b5` - 次级强调
- **Silver Glow**：`#d1d6f0` - 文本/图标

## Typography

- **Headers**：Space Grotesk Bold
- **Body Text**：Space Grotesk Medium

## Best Used For

创新发布、未来主题、夜间活动、任何希望营造沉浸式氛围的 deck。
