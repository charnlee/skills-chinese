# Modern Minimalist

干净、现代的灰阶主题，强调留白与极简排版。

## Color Palette

- **Charcoal**：`#1f1f1f` - 主背景/文本
- **Slate**：`#3f3f3f` - 次要元素
- **Cool Gray**：`#9e9e9e` - 辅助线条
- **Off-White**：`#f7f7f7` - 背景与填充

## Typography

- **Headers**：Helvetica Neue Bold
- **Body Text**：Helvetica Neue

## Best Used For

科技宣讲、现代设计方案、面向高管的汇报、强调简洁度的 deck。
