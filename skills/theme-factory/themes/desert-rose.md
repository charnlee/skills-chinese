# Desert Rose

柔和且高雅的干燥玫瑰色调，营造温暖、亲切的美感。

## Color Palette

- **Dusty Rose**：`#c58a94` - 主背景
- **Clay**：`#a05f5a` - 重点色
- **Sand**：`#f1d7c7` - 辅助底色
- **Ivory**：`#fff8f1` - 文本/浅色块

## Typography

- **Headers**：Playfair Display
- **Body Text**：Source Sans Pro

## Best Used For

品牌故事、生活方式、时尚、美妆等需要柔和、富有质感调性的演示。
